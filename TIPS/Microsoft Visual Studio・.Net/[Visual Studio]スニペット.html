スニペット(コードスニペット)とは

原義は「切れ端」や「断片」。プログラミングにおいては、再利用できるひとかたまりのコードを表す。
スニペットを作成しておくことで、よく使うコードを即座に呼び出すことができ、入力効率が向上する。ユーザ辞書と同様の機能。
Visual Studioには予め多数のスニペットが登録されている。

どのように用いるか(C#での例)


クラス定義時に付与するヘッダコメントを複数箇所で使いまわす場合、雛型をスニペットとして登録しておくと簡単に呼び出せる。

スニペットの作成

スニペットはXML形式で作成する。Visual Studio Snippet Generatorで簡単にXMLを出力できる。
以下、このサイトを用いる場合の手順を記す。

スニペットXMLの作成

  1. 上記のリンクから、Visual Studio Snippet Generatorを開く。
  2. 各項目に入力し、"Generate"を押す。
  3. 下の欄にスニペットが出力されていることを確認したら"Download"を押し、任意のフォルダにファイルをダウンロードする。

スニペットのインポート

  1. Visual Studioを開き、「ツール」>「コード スニペット マネージャー」を選択する。
  2. コード スニペット マネージャーが開くので、作成したスニペットを使用する「言語」と、格納する「場所」を選択する。
    (例 : C#で使用する場合は、「言語」に「CSharp」を選択する。)
  3. 作成したスニペットファイルを選択し、「開く」を押す。
  4. 「場所」にチェックを入れ、「完了」を押す。
  5. 「OK」を押すと、スニペットが使用可能となる。

スニペットの呼び出し

  1. スニペットを呼び出したい箇所に登録したShortcutを入力し、Tabキーを2回押す。
  2. スニペットが展開される。
