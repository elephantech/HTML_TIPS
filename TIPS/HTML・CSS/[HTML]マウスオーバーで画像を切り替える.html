マウスオーバーで画像を切り替える

<img> タグに onmouseover 、onmouseout 属性(イベントハンドラ)を追加することで、マウスオーバー(画像をポイント)時に画像を切り替えることができる。

記述例


onmouseover 、onmouseout はいずれかのみの記述でも可。但し onmouseover のみを記述した場合、マウスアウトしても画像は元に戻らない。

余談

onmouseover 、onmouseout には類似の属性 onmouseenter 、onmouseleave が存在する。
他にも onmouseup 、onmousedown 、onmousemove 、onmousewheel といった属性が存在するが、詳細は出典元を参照。
