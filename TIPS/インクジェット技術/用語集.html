RIP
Raster Image Processor
りっぷ		 印刷対象である文字や画像(主にベクタイメージ)を、ラスタイメージに変換する装置やソフトウェア。
民生プリンタにおけるプリンタドライバに相当する。
RIPの処理は大きく分けて
  • インタプリタ処理 : PS / PDFなどの元データから内容データを抽出する
  • ラスタライズ処理 : 抽出した内容データをラスタ画像化する
  • 網点化 : 画像化されたデータの各点を網点化する
から構成される。
網点
あみてん		 印刷物の濃淡を表現する点のこと。色の濃さを直接的にではなく、点の大小によって表現する。
印刷に供される色ごとに設定され、色の濃い箇所は網点が大きく、薄い箇所は小さくなる。
網点の大小(= 印刷の濃淡)を「階調」とも呼ぶ。網点の配置間隔が細かい(線数(LPI)が多い)ほど高精細(解像度が高い)となる。
この網点解像度は印刷解像度とは必ずしも等しくない点に注意。
ハーフトーンと同義。
コントーン
contone		 網点(ハーフトーン)と対になる概念。網点のような二値ではなく、色の濃さをそのまま連続的に表現したもの。
網点化により、コントーンからハーフトーンが生成される。
ハーフトーン
halftone		 ⇒ 網点
バンディング
banding		 インク・トナーの不足などにより、濃淡の縞が発生すること。
微振動
(くすぐりパルス)
tickle pulse		 インクジェットデバイスのメニスカスを移動させるために印加するパルス。
ノズルからの液体の揮発、およびノズルの乾燥を軽減できる。
ビットマップイメージ
bitmap image		 コンピュータグラフィックスにおいて、画像をドットマトリクス状のピクセル(画素)で表現する手法。
詳細はWikipediaを参照。「ラスタイメージ」と基本的には同義。
ベクタイメージ
vector image		 主にコンピュータグラフィックスにおいて、画像を円(または円弧)や直線、多角形などの解析幾何学的な「図形」の集まりとして表現する手法。
詳細はWikipediaを参照。
ベクタ形式からビットマップ形式への変換は比較的容易であるが、その逆は困難で、情報の欠損が避けられない。
メニスカス 液体と容器表面との相互作用により形成される、屈曲した液面のこと。
詳細はWikipediaを参照。
インクジェットデバイスにおいては、ノズル先端付近の液面を指す。
