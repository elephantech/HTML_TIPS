拡張メソッドとは

既存のクラス定義に手を加えることなく、インスタンスメソッドを疑似的に追加できる機構のこと。

拡張メソッドに関する留意事項

  • 拡張メソッドは静的クラス中に作成する必要がある
  • 拡張メソッドは静的メソッドとして作成する必要がある
  • 拡張メソッドが実行されるインスタンス(自分自身)を第１引数として受け取る
    (呼び出す際に渡す必要は無い)

実装例

メソッドの定義

// 拡張メソッド実装用の静的クラス static class ExtensionMethods { // 拡張メソッド(静的) // 実装先 : CheckedListBox (this 修飾の第１引数) public static void CheckAllCheckBoxes(this CheckedListBox source, bool value) { for (int i = 0; i < source.Items.Count; i++) { source.SetItemChecked(i, value); } } }

メソッドの呼び出し

checkedListBox1.CheckAllCheckBoxes(true);

余談
