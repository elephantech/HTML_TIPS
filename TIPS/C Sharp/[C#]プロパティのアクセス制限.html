目次 :
プロパティの get / set それぞれに異なるアクセス制限を設定する手順。

注意が必要な点

標準のアクセス制限(下記の例では public )を個別のアクセス制限(下記の例では private )で上書きする。
その際、「標準のアクセス制限」よりも「個別のアクセス制限」が厳しくないといけない。
(「個別のアクセス制限」を「標準のアクセス制限」よりも緩めることはできない。)

実装例

// get を public 、set を private に設定する // (標準のアクセス制限を public 、set 個別のアクセス制限を private に設定する) public string Sample01 { get; private set; } // 標準のアクセス制限を private 、set 個別のアクセス制限を protected に設定する // ⇒ 個別の制限を緩めているためコンパイルエラー private string Sample02 { get; protected set; }
