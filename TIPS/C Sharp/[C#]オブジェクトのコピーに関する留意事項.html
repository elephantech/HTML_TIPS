C#におけるオブジェクトのコピー

C#において、クラスオブジェクトの「代入」を行った場合、代入先には代入元への参照が保持される(シャローコピー)。
これはクラスオブジェクトが「参照渡し」となっているためで、コピー後に一方を更新すると、他方まで変更されてしまう。
メソッド(関数)の引数としてクラスオブジェクトを渡した場合も同様である。
(このため、オブジェクト引数をとるメソッドでは ref などの参照渡しを表すキーワードは必須ではない)
ちなみに、C/C++ではオブジェクトは「値渡し」であり、ビット単位での厳密なコピーが作成される(ディープコピー)。
また、C#においても構造体(struct)オブジェクトや、クラスオブジェクトでない組み込み型(intなど)は「値渡し」である。

シャローコピーの例

// クラス cMyClass 定義 class cMyClass { public int valA; public int valB; } class cOtherClass { public void DoSomething() { // cMyClass 型オブジェクト ob1 cMyClass ob1; ob1valA = 4; ob1valB = 5; // cMyClass 型オブジェクト ob2 に ob1 をコピー cMyClass ob2 = ob1; // ob1 、ob2 の内容を出力 ConsoleWriteLine("編集前の値"); ConsoleWriteLine("ob1valA = + ob1valA); ConsoleWriteLine("ob1valB = + ob1valB); ConsoleWriteLine("ob2valA = + ob2valA); ConsoleWriteLine("ob2valB = + ob2valB); // ob2 のメンバを編集 ob2valB = 6; // ob1 、ob2 の内容を出力 ConsoleWriteLine("編集後の値"); ConsoleWriteLine("ob1valA = + ob1valA); ConsoleWriteLine("ob1valB = + ob1valB); ConsoleWriteLine("ob2valA = + ob2valA); ConsoleWriteLine("ob2valB = + ob2valB); } }

実行結果 :

編集前の値 ob1valA = 4 ob1valB = 5 ob2valA = 4 ob2valB = 5 編集後の値 ob1valA = 4 ob1valB = 6 //< 直接編集していない ob1valB が書き換えられている ob2valA = 4 ob2valB = 6 //< 直接編集した ob2valB

MemberwiseClone() メソッドを使用したディープコピー

C#の Object 型には、ディープコピーを行うための MemberwiseClone() メソッドが定義されている。
(C#において、すべての型は Object 型から派生するため、ユーザ定義のすべての型で MemberwiseClone() を用いることができる。)
但し、MemberwiseClone() の戻り値は Object 型のため、元の型として用いるにはキャストが必要。
当然、= 演算子を用いた場合はシャローコピーとなる。

using System; using SystemCollectionsGeneric; using SystemLinq; using System; using SystemThreadingTasks; namespace CloneTest { // クローン対象のオブジェクト class CloneableItem { // クラスメンバ public int ItemId { get; set; } public string ItemName { get; set; } // ディープコピーを行うための Clone() メソッドを定義 public CloneableItem Clone() { // MemberwiseCloneメソッドを使用 // Object型で返ってくるのでキャストが必要 return (CloneableItem)MemberwiseClone(); } } // クローンのテスト class ItemCloneMain { static void Main(string[] args) { // - = - = - = - = - = - = - = - = - = - // シャローコピーの例 // - = - = - = - = - = - = - = - = - = - ConsoleWriteLine("～～～～～～～～～"); ConsoleWriteLine("○シャローコピー○"); // コピー元オブジェクト生成、値設定 CloneableItem ob1 = new CloneableItem(); ob1ItemId = 1; ob1ItemName = sword"; // = 演算子によるコピー(シャローコピー) CloneableItem ob2 = ob1; // 編集前の状態を出力 ConsoleWriteLine("編集前の値"); ConsoleWriteLine("コピー元 : + ob1ItemId + + ob1ItemName); ConsoleWriteLine("コピー先 : + ob2ItemId + + ob2ItemName); // 値を編集 ob1ItemName = knife"; // 編集後の状態を出力 ConsoleWriteLine("編集後の値"); ConsoleWriteLine("コピー元 : + ob1ItemId + + ob1ItemName); ConsoleWriteLine("コピー先 : + ob2ItemId + + ob2ItemName); // - = - = - = - = - = - = - = - = - = - // ディープコピーの例 // - = - = - = - = - = - = - = - = - = - ConsoleWriteLine("～～～～～～～～～"); ConsoleWriteLine("●ディープコピー●"); // コピー元オブジェクト生成、値設定 CloneableItem ob3 = new CloneableItem(); ob3ItemId = 3; ob3ItemName = rifle"; // Clone() メソッドの実行(ディープコピー) CloneableItem ob4 = ob3Clone(); // 編集前の状態を出力 ConsoleWriteLine("編集前の値"); ConsoleWriteLine("コピー元 : + ob3ItemId + + ob3ItemName); ConsoleWriteLine("コピー先 : + ob4ItemId + + ob4ItemName); // 値を編集 ob3ItemName = cannon"; // 編集後の状態を出力 ConsoleWriteLine("編集後の値"); ConsoleWriteLine("コピー元 : + ob3ItemId + + ob3ItemName); ConsoleWriteLine("コピー先 : + ob4ItemId + + ob4ItemName); } } }

実行結果 :

～～～～～～～～～ ○シャローコピー○ 編集前の値 コピー元 : 1 sword コピー先 : 1 sword 編集後の値 コピー元 : 1 knife // コピー元を編集すると コピー先 : 1 kinfe // コピー先にまで波及する ～～～～～～～～～ ●ディープコピー● 編集前の値 コピー元 : 3 rifle コピー先 : 3 rifle 編集後の値 コピー元 : 3 cannon // コピー元を編集しても コピー先 : 3 rifle // コピー先はそのまま
