目次 :
WEBブラウザで開いているページタイトルを簡単にコピーする手順。
Internet Explorer / Firefox / Chrome / Opera / Safari で共通。Microsoft Edge では使えない。

手順

  1. Ctrl + D を押す。
    (ブックマークへの登録画面が表示される)
  2. ページ名が選択されているので、Ctrl + C を押す。

    3. 留意事項

    上記手順はブラウザのブックマーク機能を用いる。
    すでにブックマーク済みのページであれば、ブックマークの登録名がコピーされるため、登録名を変更している場合は注意。
