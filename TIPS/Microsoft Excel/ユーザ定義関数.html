名前付き数式

目次 :
VBE(Visual Basic Editor)を用いて、独自のワークシート関数を作成することができる。

手順

  1. VBEを開く。
    (「開発」>「Visual Basic」または Alt + F11 )
  2. 関数を定義する。
    (ここでは標準モジュールを新規作成)
  3. ワークシート上で、定義した関数を入力する。

その関数が入力されたセル(自身)を取得する

Application.ThisCell メソッドを使用する。戻り値は Range 型。
セルが結合されていると失敗する点に注意。

Function MyFunction() MyFunction = Application.ThisCell.Address End Function

実行結果

はしがき

VBAの機能を用いずに、数式に名前を付けることもできる。
