作成したアプリケーションを常に管理者権限で実行させる手順

マニフェストを作成することで、アプリケーションが管理者権限を要求するようにできる。
ただし、この操作でマニフェストを「管理者権限を要求する」とした場合、
デバッグ時はVisual Studioを管理者権限で起動する必要がある。

マニフェストの作成

(プロジェクト内にマニフェストファイル(app.manifest)が存在しない場合)
プロジェクトを右クリックし、「追加」「新しい項目」「アプリケーション マニフェスト ファイル(Windowsのみ)」を選択し、「追加」をクリック。

マニフェストの編集

trustInfo > security > requestedExecutionLevel の値を下図のように書き換える。
