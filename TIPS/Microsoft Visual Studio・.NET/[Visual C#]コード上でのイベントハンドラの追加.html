イベントハンドラをコード上で追加する

ウィンドウやフォームなどのコントロールへのイベントハンドラの追加は、デザイナによる方法と、コード上から行う方法のいずれでも可能である。
(WinFormの場合は、デザイナ上で追加した場合もコードに自動的に反映されるため、結局はコードで追加することになる。)
ハンドラにはラムダ式を用いることができるため、ハンドラ用のメソッドを定義するよりも記述を簡略化できる。

手順

追加先のイベントに += でメソッド(またはラムダ式)を追加する。
ここで、ひとつのイベントに対して複数のハンドラを追加することができ、イベントの発生時には追加された(旧い)順に実行される。
