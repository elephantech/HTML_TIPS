フレキシブルプリント基板		 薄膜上の絶縁体(ベースフィルム)上に導体箔を貼り付けた回路基板。
容易に屈曲させることができ、曲げても回路特性が変化しないためヒンジ部などに用いられる他、薄いため省スペース化にも有効である。

BPP Bits Per Pixel ビット深度。ひとつの画素に何ビットの情報を割り当てるかを示したもの。画素の階調を司る。
ビットマップイメージ
bitmap image		 コンピュータグラフィックスにおいて、画像をドットマトリクス状のピクセル(画素)で表現する手法。
詳細はWikipediaを参照。「ラスタイメージ」と基本的には同義。
ベクタイメージ
vector image		 主にコンピュータグラフィックスにおいて、画像を円(または円弧)や直線、多角形などの解析幾何学的な「図形」の集まりとして表現する手法。
詳細はWikipediaを参照。
ベクタ形式からビットマップ形式への変換は比較的容易であるが、その逆は困難(または不可能)で、情報の欠損が避けられない。
ラスタイメージ
raster image		 ⇒ ビットマップイメージ
RIP りっぷ
Raster Image Processor		 印刷対象である文字や画像(主にベクタイメージ)を、印刷用のラスタイメージに変換する装置やソフトウェア。
民生プリンタにおけるプリンタドライバに相当する。
RIPの処理は大きく分けて
  • インタプリタ処理 : PS / PDFなどの元データから内容データを抽出する
  • ラスタライズ処理 : 抽出した内容データをラスタ画像化する
  • 網点化 : 画像化されたデータの各点を網点化する
から構成される。
網点 あみてん 印刷物の濃淡を表現する点のこと。色の濃さを直接的にではなく、点の大小によって表現する。
印刷に供される色ごとに設定され、色の濃い箇所は網点が大きく、薄い箇所は小さくなる。
網点の大小(= 印刷の濃淡)を「階調」とも呼ぶ。網点の配置間隔が細かい(線数(LPI)が多い)ほど高精細(解像度が高い)となる。
この網点解像度は印刷解像度とは必ずしも等しくない点に注意。
ハーフトーンと同義。「もうてん」は誤り。
コントーン
contone		 網点(ハーフトーン)と対になる概念。網点のような二値ではなく、色の濃さをそのまま連続的に表現したもの。
網点化により、コントーンからハーフトーンが生成される。
ハーフトーン
halftone		 ⇒ 網点
バンディング
banding		 インク・トナーの不足などにより、濃淡の縞が発生すること。
メディア 印刷先媒体。FPCの基材(ベースフィルム)だけでなく印画紙やコピー用紙、繊維等の総称。

めっき技術

語句
(ABC / 五十音順)		 読みまたは
正式名(略称の場合)		 意味 備考
シリンダーショック めっき面に定期的に加える振動のこと。
めっきの主反応は水素の発生を伴うが、めっき面に生じた水素の気泡をそのままにしておくと、気泡部分が正常に成膜されない(ピンホール)現象が起こる。
このため、シリンダーショックを加えて気泡の除去を行うことで、ピンホールの発生を低減させる。
電流密度 めっき面に印加する電流をめっき面の面積で割ったもの。単位面積当たりの電流を指す。
値が大きいほどめっきの形成は速くなるが、粒子は荒くなる。逆に値が小さければめっきの形成は遅く、粒子が細かくなる。
即ち、表面の平滑度と形成速度がトレードオフの関係となる。
DO 溶存酸素
dissolved oxygen		 めっき液中に含まれる酸素、およびその濃度を指す。
めっきを行うと主反応とは別に、意図しない副反応が発生することがあるが、DOを上げることで副反応を抑制できることが知られている。
このためめっき液には通常、空気をバブリングすることでDOを増やしているが、
めっき開始時にDOが高すぎると初期吸着性が悪化するため、投入前後にはバブリングの流量を下げている。
