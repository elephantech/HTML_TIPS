アドレス指定の例

コマンドライン・GUIフロントエンド(TortoiseSVNなど)で共通。

アドレス形式 コマンド例
(ルート)リポジトリ作成 ローカルパス svnadmin create \\10.14.154.94\AnyDir\SVN_Rep
インポート インポート対象 : ローカルパス
インポート先 : URL		 svn import C:\SandBox01\TargetPj file://10.14.154.94/AnyDir/SVN_Rep/TargetPj
チェックアウト チェックアウト対象 : URL
チェックアウト先 : ローカルパス		 svn checkout file://10.14.154.94/AnyDir/SVN_Rep/TargetPj C:\SandBox02\TargetPj
コミット ローカルパス svn commit C:\SandBox02\TargetPj
